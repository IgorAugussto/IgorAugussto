from REST APIs and microservices to responsive frontends and cloud infrastructure.

I work mainly with **Spring Boot** on the backend and **React + TypeScript** on the frontend,
with real hands-on experience deploying to **AWS** (EC2, S3, CloudFront), containerizing with

A full stack web application for personal financial control, featuring income, expense, and investment management with monthly projections and bank statement import.
